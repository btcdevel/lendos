Windows хостинг предназначен для сайтов использующих ASP, ASP.NET или базы данных MSSQL. Для сайта на CMS Joomla, WordPress, Drupal и других, рекомендуется использовать тарифы для популярных CMS.

Когда выбирать Windows хостинг от REG.RU

Если вам необходимы модули ASP.NET, вы используете MSSQL, БД Access работают только на Windows платформе, вы используете функции MS Sharepoint, вам необходим Windows Resource Manager.
